## рҹҡҖ Let's Get This Party Started

### рҹҺ¬ Prerequisites

Before we dive in, make sure you have:

- рҹ“Ұ **Bun 1.2+** - Fast JavaScript runtime and package manager
- рҹҰҖ **Rust 1.75+** - For backend development
- рҹҗҷ **Git** - Obviously!
- рҹҗі **Docker & Docker Compose** - For logging stack (Loki, Grafana)
- вҳ• **Coffee** - Not required but highly recommended

### рҹҺҜ Installation Speedrun

**Step 1**: Grab the code! рҹ“Ҙ

```bash
git clone https://github.com/infinitedim/portfolio.git
cd portfolio
```

**Step 2**: Install frontend dependencies! вңЁ

```bash
cd portfolio-frontend
bun install
```

**Step 3**: Set up the backend! рҹҰҖ

```bash
cd ../portfolio-backend

# Build the Rust backend
cargo build --release

# Start the logging stack (Loki + Grafana)
docker-compose -f docker-compose.logging.yml up -d
```

**Step 4**: Configure environment! рҹ”җ

```bash
# Frontend (.env.local in portfolio-frontend/)
cd ../portfolio-frontend
cp .env.example .env.local
# Edit with your values

# Backend (optional .env in portfolio-backend/)
cd ../portfolio-backend
# Set ENVIRONMENT=development, LOG_LEVEL=debug if needed
```

**Step 5**: Fire it up! рҹ”Ҙ

```bash
# Terminal 1: Start the backend
cd portfolio-backend
cargo run

# Terminal 2: Start the frontend
cd portfolio-frontend
bun run dev
```

**Step 6**: Open your browser and navigate to:

- рҹҢҗ Frontend: `http://localhost:3000`
- рҹ“Ҡ Grafana: `http://localhost:3001` (admin/admin)
- рҹ”Қ Loki API: `http://localhost:3100`

**Step 7**: Type `help` and prepare to be amazed! рҹӨҜ

### Backend Deployment

The Rust backend can be deployed using Docker:

1. **Docker Container**

   ```bash
   # Build Docker image
   docker build -t portfolio-backend .

   # Run container
   docker run -p 8080:8080 \
     -v ./logs:/app/logs \
     -v ./config:/app/config \
     portfolio-backend
   ```

   ```bash
   # Deploy Loki + Grafana stack
   docker-compose -f docker-compose.logging.yml up -d
   ```
